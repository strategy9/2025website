# Images Directory

Place your images here:

## Required Images:

1. **logo.png** or **logo.svg**
   - Strategy9 logo for navigation
   - Recommended size: 200x60px (or maintain aspect ratio)
   - Referenced in: Views/Shared/_Layout.cshtml (line ~23)

2. **hero.jpg** or **hero.png**
   - Dashboard or technology visual from Envato Elements
   - Recommended size: 1200x675px (16:9 aspect ratio)
   - Referenced in: Views/Home/Index.cshtml (line ~39)

3. **soc2-badge.png**
   - SOC2 Type 1 certification badge
   - Recommended size: 120x60px
   - Referenced in: Views/Shared/_Layout.cshtml (footer, line ~52)

## Optional Images:

- Product screenshots
- Team photos
- Additional graphics

After adding images, update the file references in the Razor views to point to the correct filenames.
